# Crucible build ideas
Navigate to https://dwolfzorn.github.io/crucible-build-ideas. The following tools are available to navigate the talent tree and create builds.
- **Talent tree search** - search talents by name/description text, tier, type, and ability.
- **Iconic spell search** - search iconic spells by name/description text, rune, gesture, and inflection.
- **Talent tree explorer** - pan/zoom view of the talent tree graph, including off-tree talents.
- **Character builder** - examples of builds for individual signature talents. These characters would likely be level 3-4.
- **Example builds** - examples of build ideas, comprised of multiple signature talents. These characters would likely be level 7-9.
